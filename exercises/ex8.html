Module 8 - Review

Inside the section with an id of answers-8, create a picture element. It should use two source elements. If the width is greater than 1200px, it should display angels-landing.jpg. If the width is greater than 1000, it should use yosemite-valley.jpg. Include an image element inside the picture element that uses arches-np.jpg as its src.

Inside the section with an id of answers-8, create an audio element. The source should be Keyboard_Cat.mp3. Be sure to give the audio element the controls attribute.

Inside the section with an id of answers-8, create a video element. The source should be Dramatic_Chipmunk.mp4. Be sure to give the video element the controls attribute and the loop attribute.
